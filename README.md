Secure ATM Banking System (Console-Based)

Python ATM simulator that supports the following: customer login, withdrawal, deposit, balance inquiry, PIN change, fund transfer, and account closure. Includes an admin module to add, delete, and unlock a user account.

Features
Customer
- Log in using Card Number + PIN
- Withdraw, Deposit & Check Balance
- Change PIN
- Transfer Money
- Close Account

- Logout
Admin
- Secure Admin Login
- Add User
- Delete User
- Unlock Locked Cards

## Technologies Applied
- Python
-JSON File Storage
-Modular Programming

## Run the Program
python main.py


##File Structure
main.py (Menu)
account.py (Customer banking operations) 
admin.py (Admin account management) 
storage.py (Read/Write JSON) 
accounts.json (User data) 
admin.json (Admin login credentials)
