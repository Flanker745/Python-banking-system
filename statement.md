# Secure ATM Banking System (Console-Based)

Problem Statement

This project simulates ATM banking, using Python for secure login and the following important banking features: withdrawal, deposit, balance inquiry, PIN change, account transfer, and account closing, all permanently stored in JSON. Admin can manage customer accounts.

##Scope
- Console-based ATM simulator
- JSON storage
- Admin and customer modules
- Secure login and error handling

##Target Users
- Bank customers (simulated)
- Bank administrators (mock)
- Students/instructors learning ATM systems

##Features
- Sign in
- Withdraw Cash
- Deposit Cash
- Transfer Money
- Balance Check
- Change PIN - Close Account 
- Admin Login 
- Add/Delete/Unlock customers
